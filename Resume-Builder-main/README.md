# 🚀 AI Resume Builder

**✨ Build Professional, ATS-Friendly Resumes Easily**

## ⚙️ Setup

### Clone Repository

```bash
git clone https://github.com/SanthoshSankar-dev/Resume-Builder.git
cd Resume-Builder
```

### Install Dependencies

```bash
npm install
```

### Run Project

```bash
npm run dev
```

Open:

```text
http://localhost:5173
```
